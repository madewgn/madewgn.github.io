Setup Bot Seller

 2023-07-08

Bahan - Bahan (Wajib):

  1. VPS untuk host bot (disarankan ububtu 22)
  2. VPS yang sudah terinstall autoscript pototo API

Setup database

  1. buat akun di deta space
  2. lalu buat database dan ambil api key nya

Cara install:

  1. upload file zip yang di berikanan oleh @madewgn ke VPS yang akan kamu gunakan untuk hosting bot nya
  2. Insta keperluan bot
apt install python3 python3-pip zip unzip -y
  1. ekstrak file zip yang di berikan
unzip namafile.zip

ganti namafile.zip dengan nama dari file zip mu

  1. install requirements
pip3 install -r req.txt
  1. ganti token dan menambahkan server

untuk token bot ada di file .env ganti saja BOT_TOKEN dengan token bot mu dan ganti juga api key di file plugins/config.py untuk nambah server baru silakan tulis di bagian create di file plugins/main.py

untuk trial nya silakan di edit di file plugins/trial.py edit yang di var l, kode callback yang digunakan adalah setelah spasi misal nama servet nya SG DO atau ID IKD nah yang di ambil itu hanya DO dan IKD nya saja, itupun sudah di tambahkan t di awal callback nya untuk menandai bahwa itu trial, t untuk trial

dan juga ganti domain nya dengan domain mu, serta server1 itu var yang di ambil dati file config.py

sesuaikan juga yang di callback.py dengan yg ada dk main.py

  1. setelah bot selesai di setup tinggal di run aja jangan lupa pakai screen agar bot tetep hidup
screen -r bot

lalu jalankan dengan perintah

python3 bot.py

Setup Userbot

userbot ini akan digunakan untuk mengatur saldo pada akun

  1. upload file zip userbot yg telah saya berikan dan di ekstark
  2. jalankan userbot nya
python3 bot.py

nah sekarang masukkan no tele yang kamu pakai, lalu masukkan juga kode otp yang di berikan oleh telegram

untuk tes userbot, ketik saja .ping
